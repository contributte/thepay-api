# contributte/thepay-api

> ⚠️ **This library is archived.** It was built for the legacy ThePay 1.0 platform (SOAP API), which has been superseded by ThePay 2.0.

## Official Library

Please use the official ThePay PHP SDK instead:

```bash
composer require thepay/api-client
```

- **Repository**: [ThePay/api-client](https://github.com/ThePay/api-client)
- **Packagist**: [thepay/api-client](https://packagist.org/packages/thepay/api-client)
- **Documentation**: [docs.thepay.eu](https://docs.thepay.eu/)

## Nette Integration

For Nette Framework integration, see [contributte/thepay](https://github.com/contributte/thepay).
